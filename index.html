Automated Meeting Minutes Generator with CrewAI + Gemini Pro

Project Overview:

An AI-driven system that transcribes .wav meeting audio, generates summaries (including sentiment and action items), and creates a Gmail draft using CrewAI agents and Gemini 1.5 Pro. The workflow automates manual documentation, reducing effort by 90% and ensuring real-time, context-aware meeting summaries.

Key Features:

  • Audio Transcription: Converts .wav files to clean, actionable text.
  • AI-Powered Summaries: Automatically generates meeting summaries, sentiment analysis, and action items.
  • Gmail Draft Creation: Seamlessly creates and sends meeting summaries to recipients via Gmail API.

Advantages:

  • 90% Reduction in Manual Documentation Time: Speeds up meeting summarization, freeing up time for more productive tasks.
  • Real-Time Results: Provides immediate access to meeting summaries and action items.
  • Scalable and Modular: Built with a configurable architecture, allowing easy customization for different workflows.
